# Metrics Definition

## Predictive Metrics
- AUROC
- AUPRC
- Accuracy
- Precision
- Recall
- F1-score

## Calibration Metrics
- Brier Score
- Calibration Curve
- Expected Calibration Error (optional)

## System Metrics
- average latency
- p95 latency
- cold start latency
- error rate
- training runtime
- estimated cloud cost

## Notes
Final metric selection depends on the finalized task definition.
